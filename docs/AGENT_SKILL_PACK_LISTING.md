I will turn your repeated workflow into a reusable AI agent skill pack with clear
instructions, trigger rules, safety boundaries, test cases, and a delivery
template. This is useful for AI automation, internal copilots, coding agents,
customer support workflows, reporting workflows, and debugging workflows.

1. What task should the agent or skill perform?
2. What input will the agent receive?
3. What output should it produce?

### FAQ

**Do you build the full app?**  
This gig focuses on reusable agent/skill packs: instructions, workflows, safety
rules, tests, and delivery templates. If you need full app development, I can
scope that separately.

**Can this work with OpenClaw, Codex, Claude-style skills, or custom agents?**  
Yes. I can write the pack in a platform-neutral format and adapt the structure
for your target runner when you provide the requirements.

**Will you handle private accounts, payments, KYC, wallets, or OAuth setup?**  
No. I can document safe steps, but those actions must be completed by the account
owner.

**Can you use my existing workflow docs?**  
Yes, as long as you provide safe, shareable material and remove secrets first.

**Do you test the packs?**  
Yes. Each pack includes test cases and a validation checklist. If implementation
testing requires your private systems, I will clearly mark that as buyer-side
validation.
