# Agent Skill Pack Custom Offer Short Pack

Use this page when a buyer asks for a custom offer on Fiverr or another service
platform. Keep the offer scoped, safe, and easy to accept.

## Fast Scope Question

```text
Thanks for the details. Before I send a custom offer, please confirm:

1. What workflow should the agent or skill pack support?
2. Which platform will run it? Examples: OpenClaw, Codex-style skills, Claude-style skills, n8n, Zapier, Make, GitHub Actions, or a custom runner.
3. Can you share one redacted sample input and the expected output?
4. What should the agent never do?

Please do not send passwords, tokens, cookies, payment details, ID documents, private customer data, or account access.
```
