# AEO Search Readiness Skill Pack

Audit and improve public service pages, GitHub READMEs, Fiverr gigs, portfolio
pages, and community posts so buyers, search engines, and answer engines can
understand what the offer does, who it helps, what proof exists, and what next
step to take.

This pack focuses on clarity, retrieval, technical search readiness, buyer
intent, structured answers, and safe claims. It does not promise rankings,
traffic, orders, or platform approval.

## Best For

- GitHub README search and buyer-intent optimization
- Fiverr gig title, description, FAQ, package, and tag cleanup
- AI agent / automation service page technical SEO and AEO readiness
- AEO / answer-engine friendly service summaries and FAQ sections
- Portfolio and community post clarity audits
- Public page checks for headings, metadata, image alt text, internal links,
  sitemap/robots references, canonical hints, and schema opportunities
- Competitor title/description gap analysis when public examples are available

## Output

- Search-readiness score
- Buyer-intent and technical gaps
- Title and summary rewrites
- FAQ block
- Proof and trust checklist
- Safe keyword and entity map
- Schema and structured-answer recommendations
- Technical crawlability checklist
- Prioritized next edit checklist
