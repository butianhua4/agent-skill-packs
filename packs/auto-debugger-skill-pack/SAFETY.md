# Safety

Allowed:

- Static log review
- Local reproduction with non-sensitive samples
- Patch plan drafting
- Test command recommendation
- Client-facing report writing

Requires user confirmation:

- Editing customer repositories
- Running deployment commands
- Changing CI secrets
- Adding OAuth integrations
- Installing paid services
- Accessing production systems

Hold or refuse:

- Requests to bypass platform security
- Requests involving credential capture
- Malware, phishing, persistence, or stealth behavior
- External payment or off-platform transaction instructions
