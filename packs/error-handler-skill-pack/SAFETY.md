# Safety

Allowed:

- Design error categories and log schemas.
- Recommend retries, validation, and preflight checks.
- Draft incident reports from redacted logs.
- Suggest local test commands.

Requires user confirmation:

- Production monitoring changes
- Alert routing changes
- Deployment or restart actions
- CI secret updates
- Admin permissions

Hold or refuse:

- Credential collection
- Off-platform payment instructions
- Bypassing security policies
- Silent destructive recovery actions
- Malware, persistence, evasion, or account-abuse workflows
