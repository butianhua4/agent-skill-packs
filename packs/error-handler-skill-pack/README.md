# Error Handler Skill Pack

Create structured error handling, logging, and recovery guidance for automation
or agent workflows.

This pack is rebuilt from the user's OpenClaw `error-handler` concept. The
source pack has useful intent but mojibake documentation, so this version is
cleaned for public, buyer-facing use.

Best for:

- Agent workflow error taxonomies
- Automation logging plans
- Node.js or PowerShell script error handling
- Client-facing incident summaries
- Repeatable debug and prevention reports

Deliverable:

- Error classification matrix
- Logging schema
- Recovery strategy
- Prevention checklist
- Client-ready incident report template
