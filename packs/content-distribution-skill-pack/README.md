# Content Distribution Skill Pack

Turn one technical asset into platform-ready launch copy for GitHub, Fiverr,
OpenClaw/community posts, X, LinkedIn, Xiaohongshu, Xianyu, and private groups.

This pack is designed for safe distribution planning. It prepares copy,
hashtags, image-card outlines, and reply hooks. It does not log in, post, spam,
bypass platform controls, or promise traffic.

## Best For

- Turning a GitHub release into social posts
- Rewriting Fiverr gig proof into buyer-facing launch copy
- Creating Xiaohongshu or Xianyu post drafts from a service package
- Preparing OpenClaw/community announcements for new skill packs
- Building repeatable daily distribution from one proof asset

## Output

- Platform-specific post drafts
- 9-card Xiaohongshu outline
- X/LinkedIn short posts
- Fiverr update snippet
- Community announcement
- Reply hooks and follow-up prompts
