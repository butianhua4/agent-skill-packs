# Offer Builder Skill Pack

Turn a market opportunity, buyer pain, or validated demo into a sellable service
package with title, pricing ladder, buyer requirements, proof asset, delivery
scope, and upgrade path.

This pack is built for AI agent, automation, workflow-debug, CI, skill-pack, and
technical delivery services. It does not make legal, tax, payout, or revenue
guarantees.

## Best For

- Turning market-radar output into a Fiverr gig
- Creating 99 / 299 / 599 USD service tiers
- Packaging Chinese 299 / 999 / 1999+ RMB offers
- Converting a GitHub demo into a buyer-ready service
- Scoping fixed-price work before sending a custom offer

## Output

- Offer positioning
- Package ladder
- Buyer requirements
- Delivery scope and exclusions
- Proof asset checklist
- FAQ and objection handling
- Launch next action
