# Agent Skill Pack Auditor

Audit an existing folder of agent skills and turn it into a clean, publishable,
testable skill-pack library.

Best for:

- OpenClaw skill folders
- Codex-style skills
- Prompt packs
- Agent operating manuals
- Automation playbooks

Deliverable:

- Inventory
- Risk classification
- Cleanup plan
- Pack standardization checklist
- Migration-ready output
