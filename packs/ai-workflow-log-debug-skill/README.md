# AI Workflow Log Debug Skill

Diagnose failed AI agent, CI, or automation logs and produce a clear fix report.

Best for:

- Failed agent tool calls
- Broken JSON or structured output
- CI logs with unclear failures
- n8n, Make, Zapier, or custom Node.js automation failures

Deliverable:

- Root cause summary
- Risk level
- Fix plan
- Reproduction steps
- Client-ready report
