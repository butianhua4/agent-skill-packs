# Safety

Allowed:

- Analyze pasted or uploaded logs.
- Work with redacted examples.
- Suggest safe reproduction steps.
- Produce patch plans and reports.

Requires user action:

- Login
- Payment
- KYC
- Admin permission
- OAuth authorization
- Production deployment
- Opening unknown external links

Refuse or redirect:

- Credential collection
- Payment outside the platform
- Malware, phishing, or account abuse
- Bypassing platform security checks
