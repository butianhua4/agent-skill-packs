# Progress Reporter Skill Pack

Turn technical work logs into clear buyer-facing progress updates.

This pack helps an agent summarize what changed, what was verified, what is still
blocked, and what should happen next. It is designed for AI automation, CI,
debugging, and implementation services where the buyer needs confidence without
reading raw logs.

## Best For

- Fiverr or freelance delivery updates.
- GitHub issue or PR status summaries.
- AI agent debugging work where each step needs a buyer-readable report.
- Multi-step implementation work with risks, blockers, and verification notes.

## Output

- Short status update.
- Detailed progress report.
- Risk and blocker list.
- Verification summary.
- Next-step recommendation.

## Not For

- Faking progress or claiming unverified work.
- Reporting private credentials, personal data, or buyer secrets.
- Replacing platform order messages or payment workflows.
