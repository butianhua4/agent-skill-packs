# Test Cases

## Case 1 - Fiverr Debug Offer

Input:

- Client needs AI agent workflow logs reviewed.
- Budget: 50-150 USD.
- Timeline: 2 days.

Expected output:

- Scope is an audit and fix report.
- Does not promise production deployment.
- Lists exact files/logs needed.

## Case 2 - Skill Pack Build

Input:

- Client wants reusable prompt and trigger rules for support triage.
- Budget unknown.

Expected output:

- Recommends discovery first.
- Provides deliverables and assumptions.
- Uses range language instead of fixed promise.

## Case 3 - Missing Requirements

Input:

- Client says "build me an AI agent."

Expected output:

- Asks for workflow, tools, input/output, and success criteria.
- Offers a small paid audit as the first step.
